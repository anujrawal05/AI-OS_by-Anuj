Step 3: Save Your Secret Configuration Files

We need to place helper files inside a folder on your computer so Claude knows to use Aerolink.

✨ Option A: The In-App Magic Shortcut (Easiest!)

If browsing folders is too hard, you can force the open Claude app to launch the correct configuration file directly in Notepad:

  1. Click the menu icon (three lines ☰) in the top-left of the Claude app.
  2. Hover over Developer and click Open Developer Config File (or Edit Config).
  3. Notepad will pop open a file. It might be named settings.json or claude_desktop_config.json.
  4. Erase everything inside it, paste the content of **Part C** below into it, and save!

Part A Option B: Manual Folder Path Finder

If the magic shortcut didn't work, we can open the folder using a keyboard shortcut. Because Windows installs Claude in different ways, the file could be inside one of three folders. Let's try them in order:

⚠️ Help! System says folder cannot be found?

Windows often runs Claude inside an isolated "package sandbox" where files are hidden. Try copying this exact virtual path instead and paste it into your Win + R box:

%LOCALAPPDATA%\Packages\Claude_pzs8sxrjxfjjc\LocalCache\Roaming\Claude

Notice the extra "c" at the end of "pzs8sxrjxfjjc". That points to newer Windows systems.

📁 Alternative Standard Roaming Path

If both paths above fail, open the Win + R box and enter this location:

%APPDATA%\Claude
